# kobo-calendar
將 kobo 每個禮拜的 99 清單列出來，並加上相對應得 ics 檔案  
使用 ics 可以將該天特價內容加入至行事曆中 (ios 可直接使用)  
使用 google calendar 可以直接加入 google 行事曆，不用額外下載  
在使用提醒功能時，請檢查各項是否正確  
另外，有電腦或程式相關的開箱/問題/心得，歡迎分享至無廣告論壇 [Community TW](https://community.tw)
